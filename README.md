# superdogecoinbot
Bot I made for superdoge.com Official Telegram Group

## Dependencies
- requests (Should be installed by default in most of the IDEs)
- pyTelegramBotAPI
- Pillow
- qrcode
- Flask

```
pip install requests
```

```
pip install pyTelegrambotAPI
```

```
pip install Pillow
```

```
pip install qrcode
```

```
pip install Flask
```
